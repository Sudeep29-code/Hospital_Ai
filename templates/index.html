About Smart Hospital AI

Smart Hospital AI is an intelligent queue management system designed to reduce hospital waiting time using AI-based patient prioritization.

The system sorts patients based on their condition, age, and department to ensure critical cases are handled first.

Built using Python Flask, Machine Learning, HTML, and CSS, this project aims to modernize hospital management systems.
